# 西游修仙界三大势力设定

## 一、天庭

### 势力概述
**起源**：自鸿蒙初判，盘古开天，清气上浮为天。道祖太上老君于三十六天外立"弥罗宫"，传下道统。后历经龙汉、赤明、开皇、延康、上皇五劫，诸天星宿、山川神灵渐次归位，共尊**昊天金阙无上至尊自然妙有弥罗至真玉皇上帝**（即玉帝）为三界共主，筑天庭于三十三天之**凌霄宝殿**，统御万天，维系三界秩序。

**历史**：天庭权威历经数次挑战。上古时期，巫妖大战，天庭初立，根基未稳。封神一役，道门三教借人间王朝更迭，册封三百六十五位正神，充实天庭班底，确立了以玉帝为核心的"神道"体系，然亦埋下截教门人（多为妖族出身）与天庭旧神的矛盾。西游量劫，佛门东渐，天庭与佛门合作，借取经之事重塑三界格局，亦暗含制衡。

**结构**：天庭乃高度中央集权的官僚化神权组织。以**玉帝**为至高主宰，下设**四御**辅佐。行政体系分"文武"二途：
- **文官系统**：以**文昌星君**、**太白金星**为首，掌管仙籍、功德、文书、礼仪、风雨雷电等日常运转。
- **武官系统**：以**托塔李天王**（降魔大元帅）、**哪吒三太子**（三坛海会大神）为帅，统辖**四大天王**、**二十八宿**、**九曜星官**、**三十六天罡将**、**七十二地煞君**及十万天兵天将，司职征伐、镇守。
另有**雷部**、**火部**、**瘟部**、**斗部**等神部，各司其职。

### 核心权力架构与职位
- **至高神**：玉皇大帝（总执天道，三界之主）
- **四御**：
    1. 中天紫微北极大帝（协助玉帝，执掌天经地纬、日月星辰）
    2. 勾陈上宫天皇大帝（协助玉帝，执掌南北极与天地人三才，统御万雷）
    3. 南极长生大帝（协助玉帝，执掌人间寿夭祸福）
    4. 承天效法后土皇地祇（协助玉帝，执掌大地山川、万物生长）
- **三清道祖**（超然地位，不直接理政）：元始天尊、灵宝天尊、道德天尊（太上老君）。

### 代表人物
1. **玉皇大帝**：境界**混元大罗金仙**（道门果位，近乎天道化身）。来历：历经一千七百五十劫，每劫十二万九千六百年，修成无极大道。
2. **太上老君（道德天尊）**：境界**混元无极太上大道君**（三清之一，万法之源）。来历：道之化身，曾化身老子传道人间，为炼丹、炼器之祖。
3. **托塔天王 李靖**：境界**大罗金仙**。来历：曾师从度厄真人，后得天庭敕封，掌管天兵天将，有法宝**玲珑宝塔**。
4. **三坛海会大神 哪吒**：境界**太乙金仙**（战力可比大罗）。来历：灵珠子转世，太乙真人弟子，莲花化身，神通广大。
5. **显圣二郎真君 杨戬**：境界**大罗金仙**。来历：玉帝外甥，师从玉鼎真人，肉身成圣，听调不听宣，坐镇灌江口，有**七十三变**、**八九玄功**。

### 核心矛盾与内部派系
- **帝权与神权**：玉帝的行政权威与三清（尤其是老君）的道统权威之间存在微妙平衡。老君的兜率宫宛如国中之国。
- **新旧势力**：封神榜上之神（多为魂魄受封，修为受限，如雷部诸神）与肉身成圣或自行修炼上天的仙人（如二郎神、哪吒）之间存在隔阂与竞争。
- **实权派与清贵派**：手握兵权的李天王系、执掌实务的部门，与一些清闲高贵的古仙、星君之间存在资源与话语权的争夺。
- **招安派与清剿派**：对下界妖魔，以太白金星为代表的派系主张"招安"，而以李天王为代表的武将派系则倾向"武力清剿"。

### 与其他势力的关系
- **与佛门**：**战略合作，互有制衡**。西游量劫是双方高层协议的结果，旨在共同应对劫数，并重新划分势力范围。
- **与妖界**：**上位者与潜在颠覆者**。天庭视妖界为"化外之民"、"祸乱之源"，采取**清剿、招安、利用**并用的策略。

## 二、佛门（西方教/佛教）

### 势力概述
**起源**：源自上古西方教，由**接引道人**（后为阿弥陀佛）、**准提道人**（后为准提菩萨）二位圣人创立。封神之战后，西方教大兴，演化为**佛教**，立**极乐世界**于西牛贺洲灵山大雷音寺。以**如来佛祖**（多宝道人化胡为佛）为现在世掌教尊，开创全新格局。

**历史**：封神时，西方二圣广度有缘人入西方，奠定根基。如来掌教后，整合教义，确立"佛陀-菩萨-罗汉"果位体系，势力急剧膨胀。为传播信仰，化解劫难，发起"西天取经"工程，主动向东胜神洲、南赡部洲扩张。

**结构**：佛门是以**信仰**和**修行果位**为核心的宗教团体。组织结构以灵山为核心，分为：
- **核心圣境**：灵山大雷音寺（如来道场）、珞伽山紫竹林（观音道场）、五台山（文殊道场）、峨眉山（普贤道场）等。
- **果位等级**：**佛**（自觉、觉他、觉行圆满） > **菩萨**（觉有情，上求佛道，下化众生） > **罗汉**（自觉，破我执） > **揭谛**、**金刚**、**力士**、**比丘**等。
- **四大菩萨**：观音（大悲）、文殊（大智）、普贤（大行）、地藏（大愿）地位崇高，各有愿力与道场。

### 核心权力架构与职位
- **横三世佛**：
    1. **中央娑婆世界释迦牟尼佛（如来佛祖）**：现在世掌教，最高领袖。
    2. **东方净琉璃世界药师佛**：主管东方，发十二大愿。
    3. **西方极乐世界阿弥陀佛**：接引佛，过去佛，地位超然。
- **竖三世佛**：燃灯古佛（过去）、释迦牟尼佛（现在）、弥勒佛（未来）。

### 代表人物
1. **如来佛祖（释迦牟尼）**：境界**混元大罗金仙**（佛门果位，等同道门混元）。来历：原为截教大师兄多宝道人，化胡为佛。
2. **观音菩萨（观世音）**：境界**大罗金仙**（佛门果位）。来历：曾为阐教慈航道人，持玉净瓶、杨柳枝，有**七十二相**，寻声救苦。
3. **文殊菩萨**：境界**大罗金仙**。来历：曾为阐教文殊广法天尊，司大智慧，骑青狮，持慧剑。
4. **大日如来/乌巢禅师**：境界**大罗金仙巅峰**。来历：神秘存在，曾传《心经》于取经人。
5. **斗战胜佛 孙悟空**：境界**大罗金仙**（成佛后）。来历：天生石猴，师从菩提祖师，大闹天宫后被压五行山，经取经之路，终成正果。

### 核心矛盾与内部派系
- **修行理念之争**：存在"渡己"（追求自我解脱的罗汉道）与"渡人"（发宏愿度尽众生的菩萨道）之间的侧重差异。
- **出世与入世**：部分古佛、菩萨偏向灵山静修，而以观音为代表的势力则积极介入世俗。
- **未来接班暗流**：弥勒佛作为未来佛，其势力与如来当前核心团队之间可能存在微妙关系。

### 与其他势力的关系
- **与天庭**：**战略盟友，隐性竞争**。共同维护三界大框架，在信仰传播、人才吸纳、势力范围上存在竞争。
- **与妖界**：**主要的度化与镇压对象**。采取 **"有缘则度，为恶则镇"** 的策略。

## 三、妖界

### 势力概述
**起源**：与天地同生。鸿蒙开辟后，天地灵气滋养万物，飞禽走兽、草木精怪乃至顽石，机缘巧合下开启灵智，吸纳日月精华、天地灵气修炼成形，即为"妖"。

**历史**：上古巫妖大战，妖族掌天，巫族掌地，此为妖族最辉煌亦最惨烈的时代。战后，妖族天庭崩毁，妖族沦为散流。此后亿万年，妖族再无统一皇者，进入"**万妖割据**"时代。

**结构**：**无中央政权，呈绝对散沙状**。以**山头、洞府、水域**为单位，由一个个**妖王**自治。妖界遵循最原始的**丛林法则**，弱肉强食，强者为尊。

### 核心权力架构（无固定架构，以实力强弱论）
- **顶尖妖圣/大圣**：实力可比肩大罗金仙，割据一方（如牛魔王、孙悟空曾为七大圣）。
- **一方妖王**：占据名山大川，统领数万至数十万妖众，实力在天仙至太乙金仙不等。
- **洞主、寨主**：占据小型洞府，实力多为地仙、天仙。
- **散妖、小妖**：无固定势力，或依附大妖，或独自修行。

### 代表人物
1. **平天大圣 牛魔王**：境界**大罗金仙**。来历：上古异种奎牛后裔，修炼万载，交友广阔，为妖界无冕之王。
2. **混天大圣 鹏魔王**：境界**太乙金仙巅峰**（速度冠绝妖界）。来历：可能与凤凰或大鹏血脉有关，振翅九万里。
3. **通风大圣 猕猴王**：境界**太乙金仙**。来历：灵明石猴之外的另一异种神猴，耳聪目明，消息灵通。
4. **白骨夫人（白骨精）**：境界**地仙巅峰**。来历：一具白骨受日月精华成妖，无根无底，代表草根妖怪艰难求存的典型。
5. **九头虫**：境界**太乙金仙**。来历：上古凶兽血脉，入赘乱石山碧波潭万圣龙王家，盗取佛宝。

### 核心矛盾与内部派系
- **生存危机下的各自为战**：面对天庭、佛门的外部压力，妖界无法形成统一战线。
- **正统与草根的对立**：拥有上古血脉、传承完整的"大妖"与普通生灵修炼成精的"草根妖怪"之间存在鄙视链。
- **理念分歧**：是逍遥自在占山为王，还是追求长生大道？是接受招安或度化，还是彻底对抗？
- **资源争夺**：灵脉、洞天福地、天才地宝等资源的争夺，是妖界内部永恒的主题。

### 与其他势力的关系
- **与天庭**：**被镇压与反抗的关系**。天庭是妖界最主要的剿灭者与管理者。大部分妖怪视天兵天将为天敌。
- **与佛门**：**被度化与利用的关系**。佛门对妖界态度更为复杂：为恶者镇压，有缘者度化，甚至直接收为坐骑、护法。
- **内部关系**：**弱肉强食，合纵连横**。妖王之间今日为友，明日为敌是常态。
