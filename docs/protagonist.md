# 主角人物卡

## 基础信息
- **姓名**：林渊（化名）/ 林玄（真名）
- **年龄**：17岁（开篇）
- **身份**：菩提祖师座下圆字辈弟子 / 天藏之体拥有者

## 外貌
银白短发，左眼漆黑如墨、右眼淡金隐现异芒。面容清隽冷峻，身形瘦削却骨架硬朗。常年穿青灰色长衫，腰悬古朴铜钱。

## 性格
表面沉默寡言，实则心思缜密、重情重义。因幼年经历对"信任"敏感，却能在关键时刻为所爱之人赴死。隐忍但不懦弱，狠辣却有底线。

## 背景
林家曾是守护"天藏秘境"的隐世家族，八年前惨遭灭门。年幼林玄被忠仆以命换命送出，抹去记忆流落市井。体内封印着天藏秘境的钥匙——「万藏图」。

## 金手指
**「万藏图」** + **「双瞳异化」**

1. **探藏**：感知方圆十里内隐藏的功法、秘境、宝物
2. **纳藏**：将物品封入图中空间，时间静止
3. **解藏**：解析功法本质，推演完善或创造新法
4. **双瞳**：左眼现世，右眼窥平行宇宙

## 成长路线
炼体期 → 藏气期 → 藏意期 → 藏魂期 → 藏天期 → 万藏归一

## 命运节点
- 穿越：成为孙悟空小师弟
- 觉醒：窥见悟空被压五行山命运
- 抉择：改变命运还是顺应天道
- 冲突：与大圣理念分歧
- 觉悟：定数之外尚有一线生机
