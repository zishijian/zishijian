# 《我是大圣小师弟》创作资料

## 第一章开头

头好痛……

林渊艰难地睁开双眼，入目是一片苍翠古松，云雾缭绕。

等等，这是哪？

他猛地低头，发现自己竟盘膝坐在青石台阶之上，身上穿着一件粗布麻衣，双手还保持着打坐的姿势。

"轰！"

一股陌生的记忆如潮水般涌入脑海——

方寸山，斜月三星洞，菩提祖师……

穿越了？

而且还是《西游记》的世界！

"新来的，发什么呆呢？"

一道声音从身后传来，林渊回头，瞳孔骤然收缩。

只见一只金毛猴子正蹲在台阶上，抓耳挠腮地看着他，眼中满是灵动狡黠。

那张脸，分明就是——

"师、师兄？"

猴子咧嘴一笑："算你小子识相，从今往后，俺老孙罩着你！"

林渊心脏狂跳。

齐天大圣孙悟空，竟然是他师兄！

就在这时，一道苍老而威严的声音从山顶传来：

"悟空，带新弟子上来。"

猴子脸色一变，拉着林渊就往山上跑："快走，师父要见你！"
