# 菩提老祖 · 人物小传

## 【真实身份】

道号菩提老祖，亦称须菩提。然此名非其本名，乃他自斩真名后所戴之面具。

其真实身份，是上一纪元"天道"崩塌时唯一逃出法则清算的存在——旧世的**"执笔者"**。他曾书写万物命格，掌因果之笔，定轮回之序。天道崩塌之际，他亲手抹去了自己在一切典籍中的痕迹，以"不存在"之身存活至今。

## 【隐居方寸山】

方寸山并非凡山，而是旧天道崩裂时留下的一道"裂缝"——介于新旧两界之间，不在三界五行之内。他藏身于此，既为躲避新天道的感知，也为**守护裂缝中残存的旧世法则碎片**。

斜月三星洞的每一面墙壁，都刻满了被世人遗忘的旧世文字。灵台方寸，心即牢笼。

## 【九辈弟子之限】

他只收九辈弟子，因"九"为旧天道的极数。每一代弟子，皆是他布下的**一粒棋子**——有人被送入天庭，有人被投入妖域，有人散落人间。

九子归位之时，便是旧法则重铸之日。前八辈弟子或死或散，无人知晓自己真正的使命。他从不悲悯，因为执笔者眼中，众生皆为字。

## 【对主角的关注】

主角是第九辈弟子，却也是唯一的**"异数"**——他的命格在菩提推演中始终模糊不清，像一页被水洇湿的残稿。更令菩提忌惮的是，主角身上携带着某种连旧天道都不曾记载的气息。

菩提收他为徒，名为传道，实为**试探**——他想知道，主角究竟是来修补这个世界的，还是来彻底终结它的。
