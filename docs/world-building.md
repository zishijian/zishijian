# 藏子世间 · 世界观设定集

> 「灵台方寸山，斜月三星洞」—— 此地非他处，乃是修心之地

---

## 一、核心设定

### 1.1 圆字辈特殊能力：【圆照】

圆字辈弟子结"圆印"，即可内视己身、外察天地。

> 短暂将五感延展至方圆一里，如掌上观纹，勘破虚妄、预判先机。
> 
> *代价：精神消耗巨大，过度使用会损伤神识。*

### 1.2 觉字机缘本质：【觉念】

非外物赐予，而是识海中"沉睡真灵"的瞬间觉醒。

> 受特定情绪或因果牵引，真灵破茧，赋予宿主改写局部现实的权限。
> 
> *代价：记忆碎片化，每一次觉醒都会失去部分过往。*

### 1.3 主角异能：双瞳异化

主角天生"重瞳"，实为贯通两界之隙。

| 左瞳 | 右瞳 |
|:---:|:---:|
| 映现当前宇宙的真实 | 倒映平行宇宙的可能 |

> 因幼时濒死体验，两瞳短暂融合，使他能主动窥视并踏入时间线的分岔点。

## 二、方寸山七层境界

方寸山斜月三星洞，菩提祖师传道之所。洞府分七层，对应修行之七个阶段。

### 第一层：心启（入门之基）

初入山门，洗净凡尘。明心见性，识己本真。

- 修行者需在山门前叩首七日
- 通过心性考验方可入内
- 此层设有"问心石"，可映照修行者内心执念

### 第二层：气引（感气入门）

引气入体，初窥仙途。感天地之气，纳为己用。

- 学习吐纳之法
- 感应周身灵气流动
- 可进行基础法术启蒙

### 第三层：形淬（淬炼肉身）

以气淬体，锤炼筋骨。身如精钢，方堪问道。

- 修习基础炼体功法
- 肉身强度大幅提升
- 为后续法术修行奠定基础

### 第四层：法通（法术通玄）

法术初成，变化万千。阴阳五行，皆可驱用。

- 可修习基础法术
- 七十二变入门功法开放
- 掌握至少一门护身神通

### 第五层：意凝（意志凝练）

心意合一，念动法随。意志坚定，诸邪不侵。

- 精神力大幅提升
- 可抵御心魔侵蚀
- 悟字辈弟子所在层次

### 第六层：神游（元神出游）

元神出窍，神游太虚。千里之遥，瞬息可至。

- 元神可短暂脱离肉身
- 感知范围扩展至方圆百里
- 圆字辈弟子所在层次

### 第七层：觉空（大觉圆满）

觉悟本空，返璞归真。真灵觉醒，触及本源。

- 识海真灵完全觉醒
- 可改写局部现实规则
- 觉字辈弟子所在层次，亦是传说中菩提祖师所处的境界

## 三、九辈分体系

菩提祖师座下弟子无数，按入门顺序分为九辈，象征修行之九个境界。

```
辈分顺序：广 → 大 → 智 → 慧 → 真 → 悟 → 如 → 圆 → 觉
```

| 辈分 | 代表人物 | 境界 | 说明 |
|:---:|:---:|:---:|:---|
| 广辈 | 历代首徒 | 凡境巅峰 | 开山弟子，传承道统 |
| 大辈 | 历代护法 | 灵境初阶 | 守护山门，传授新徒 |
| 智辈 | 教习师兄 | 灵境中阶 | 负责日常修行指导 |
| 慧辈 | 典籍管理者 | 灵境高阶 | 掌管洞府典籍秘法 |
| 真辈 | 修行有成者 | 仙境初阶 | 已证长生，可称真仙 |
| 悟辈 | 大神通者 | 仙境高阶 | 如孙悟空，得七十二变、筋斗云 |
| 如辈 | 隐世高人 | 神境初阶 | 行事如风，来去无踪 |
| 圆辈 | 大觉之前 | 神境中阶 | 可内视己身、外察天地 |
| 觉辈 | 觉字机缘 | 神境圆满 | 识海真灵觉醒，触及本源 |

### 觉字辈特别说明

> 觉字辈是最后的辈分，亦是菩提祖师最看重的一辈。
> 
> 传说集齐九十九位"觉"字辈弟子，便可打开通往永恒的大门。
> 
> 然而无数岁月过去，觉字辈始终未能圆满。
> 
> 直到主角的出现——这位来自地球的年轻人，承载着"觉字机缘"的命运。

## 四、金手指：逆道真解

主角所持之特殊能力，超越常规修行体系的逆天之法。

### 来历

据传为混沌初开时，某位不愿成圣的绝世强者所创，后流转至菩提祖师手中，代代相传，只待有缘人。

### 核心特性

| 特性 | 说明 |
|:---:|:---|
| 逆行伐道 | 可越阶挑战，不受境界压制 |
| 时空窥探 | 短暂窥视过去未来片段 |
| 因果改写 | 局部改写因果律，代价极大 |

### 使用代价

> 每次使用都会加速"觉"字机缘的消耗
> 
> 使用过度将导致记忆永久丧失
> 
> 最严重时，可能失去自我意识

### 修炼口诀

```
逆天而行道自通，
真解藏在有无中。
不求长生不老术，
但愿今朝破苍穹。
```

## 六、方寸山地理志

### 山门

方寸山山门无守卫，开放以待有缘人。门旁有古松一棵，枝干虬劲，四季常青。

### 斜月三星洞

洞府之内，陈设皆为天然石器——石床、石凳、石盆、石碗，无金玉装饰，契合"清静无为"之理念。

洞顶有斜月投影，三点星光洒落，昼夜轮转自有其时。

### 七层灵塔

洞府深处有一七层灵塔，每层各有妙用：

- **第一层**：藏书阁，藏有万卷经典
- **第二层**：演武场，供弟子切磋
- **第三层**：丹房，炼制丹药
- **第四层**：炼器室，锻造法器
- **第五层**：闭关室，高阶弟子修行所用
- **第六层**：历代祖师牌位
- **第七层**：祖师清修之地，禁地

### 隐秘空间

洞中另有隐秘空间，独立于三界之外，时间流速与外界截然不同。外界一日，洞中或已一年。

## 七、世界观核心概念

### 灵韵

天地间流动的根本能量，一切修行者汲取的力量源泉。

- 分布于山川河岳之中
- 灵韵浓郁之处可称"洞天福地"
- 方寸山即为灵韵极为浓郁的修行圣地

### 真灵

每个生命识海深处的本源，包含着无限可能。

- 真灵沉睡者为凡人
- 真灵觉醒者踏入修行
- 真灵完全觉醒者，可触及永恒

### 因果

世间万物的联系网络，牵一发而动全身。

- 修行者因修行深浅，对因果的感知能力不同
- 觉字辈可短暂窥探因果
- 逆道真解可局部改写因果
