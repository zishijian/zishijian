# 第一章 世界观背景

## 一、机缘起源：混沌初开的命运织机

太初有觉，觉生混沌。当第一缕「知」的念头在无垠虚空中泛起涟漪，盘古未醒的鸿蒙里，便有了「机」的震颤与「缘」的丝线。那不是创造，而是**觉醒**——如同晨露在叶尖将坠未坠时，忽然映照出整个宇宙的倒影。

混沌如未染的素绢，而「觉」是第一滴墨。墨滴晕散的轨迹，便是命运最初的法则：**所有可能性同时存在，又彼此消隐**。正如《道德经》所言「道生一，一生二」，那滴墨里藏着三千世界的倒影，每一次震颤都让某个世界凝实，又让另一个世界褪为残影。

这便是「机缘」的本质：不是被书写好的命书，而是**正在呼吸的织机**。每个生灵的觉醒，都为织机添上一根纬线；每次呼吸的停顿，都让某段图案永远失落。

## 二、缘分法则：五行生克间的叹息

缘分如风，看不见却吹动万物。其流转遵循三重交织的律法：

**五行之气**是缘分的呼吸。金生水时如钟磬余音，水生木时若春雨润芽，木生火时似萤火初燃，火生土时若落日熔金，土生金时如矿脉低鸣。而相克处藏着物哀之美——金克木是刀刃与年轮的告别，木克土是根须穿透陶俑的温柔暴力。

**十二生肖**是缘分的时辰。子鼠衔着前世未说出口的承诺，丑牛踏碎轮回中相似的月光。三合局是灵魂久别重逢的震颤，六冲局则是转身时衣袖擦过的微凉。

**二十八星宿**是缘分的经纬。苍龙七宿主生发之缘，朱雀七宿掌炽烈之遇，白虎七宿司斩断之机，玄武七宿守静默之契。当某人的命星移入新的宿度，便会在街角闻到似曾相识的桂花香。

## 三、命运之轮：周行不殆的露与樱

命运不是直线，而是**年轮般的漩涡**。外层是可见的因果循环：春种秋收，月盈则亏，如同樱花必然在盛开七日后凋零。中层是暗涌的宿命回音——此刻落下的雨，或许正是百年前某滴眼泪的归来。

最深处是「觉」的试炼场：**每一次轮回，织机都会留下相同的空白处**。有人再次跌入同样的执念，有人却在空白处画出新的纹路。这便是「物哀」的终极启示：明知露水将逝仍折射朝阳，明知樱花将落仍开到极致。

当命运之轮转过三千六百周天，所有丝线终将汇成一句话：**「你所遇见的，都是久别重逢；你所错过的，都是为了此刻的觉醒。」** 而织机永远在晨雾中轻响，等待下一个睁开眼睛的织梦人。

# 第二章 核心玩法系统

## 一、能量积累系统

### 1.1 双生能量池

| 能量类型 | 基础上限 | 溢出规则 | 每日基础获取 |
|---------|---------|---------|------------|
| **福运值** | 10,000点 | 溢出部分的20%转化为下期初始值（上限2,000点） | 300点 |
| **衰运值** | 10,000点 | 无溢出转化，达上限后触发"厄运反噬"事件 | 150点 |

### 1.2 能量交互规则

- **福衰转化**：每消耗1点福运可抵消2点衰运（1:2比例）
- **动态平衡**：当福运值＞衰运值时，每日衰运获取+50%；反之福运获取+30%
- **特殊日晷**：现实节气/节日当天，福运获取效率×200%

### 1.3 每日自然恢复公式

```
E_base = 120 + (玩家等级 × 2)
```

*（等级上限60级，满级自然恢复240点/日）*

### 1.4 主动获取途径

| 来源 | 公式/固定值 | 每日上限 |
|------|-------------|----------|
| 禅修打坐（每分钟） | `0.5 + 禅修等级×0.1` | 60点 |
| 秘境探索（每次） | `15~35随机` | 150点 |
| 机缘事件触发 | `20~50随机` | 100点 |
| 好友互助 | `5×好友数量` | 50点 |

### 1.5 特殊状态触发

| 状态名称 | 触发条件 | 效果持续 |
|---------|---------|---------|
| **吉星高照** | 福运值≥8,000且连续3日增长 | 24小时：所有概率判定+15% |
| **流年不利** | 衰运值≥7,000或单日衰运增长＞500 | 12小时：商店物价+30%，掉落率-20% |
| **阴阳调和** | 福运与衰运差值≤500 | 6小时：解锁特殊合成配方 |
| **气运逆转** | 福运值从＜1,000单日增至＞3,000 | 立即重置所有能力冷却时间 |

### 1.6 收支平衡表（以满级玩家为例）

```
日均总获取：
自然恢复: 240
禅修打坐: 60 (上限)

### 2.2 动态概率算法

```
def 计算实际掉率(累计未出高稀有度次数):
    基础概率 = 对应稀有度基础掉率
    if 累计次数 > 保底阈值×0.7:

## 四、数值校验公式

### 4.1 经济循环健康度指标

```
能量循环率 = (日均消耗/日均获取) × 100%
目标区间：80%-120%

道具获取满意度 = (实际掉率/预期掉率) × 保底系数
目标值：0.9-1.1

成就推进速度 = (完成成就数/总成就数) / 游戏天数
目标曲线：前30天斜率0.05，30-90天斜率0.02，90天后斜率0.005
```

### 4.2 动态调整规则

1. 当某稀有度实际掉率连续3天低于预期值15%时，触发补偿机制
2. 能量盈余超过3天储备量时，开放限时高消耗玩法
3. 成就完成率偏离预设值20%时，启动难度动态调整
